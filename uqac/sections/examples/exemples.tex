\subsection*{Références bibliographiques}

Pour citer une référence, utilisez \texttt{\textbackslash cite\{clé\}}. Par exemple : "Selon \cite{exemple}, les méthodes numériques..."

Pour ajouter une référence dans \texttt{references.bib} :
\begin{verbatim}
@article{exemple,
  author = {Nom, Prénom},
  title = {Titre de l'article},
  journal = {Nom du journal},
  year = {2025}
}
\end{verbatim}
